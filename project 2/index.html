History of HTML

What is HTML?

HTML (HyperText Markup Language) is the fundamental building block of the Web. It provides structure to web content through a system of elements and tags.

DOM Structure

HTML → HEAD → TITLE

HTML → BODY → H1

HTML → BODY → P

HTML History Timeline

1991 - HTML Birth

Tim Berners-Lee creates HTML at CERN

1995 - HTML 2.0

First official standard by IETF

1997 - HTML 3.2

W3C recommendation with tables and math

2014 - HTML5

Modern semantic elements and APIs

The Founder

Sir Tim Berners-Lee

British computer scientist who invented HTML while working at CERN. Later founded W3C to oversee Web development.
